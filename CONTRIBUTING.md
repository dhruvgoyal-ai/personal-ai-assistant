# Contributing Guidelines

Thank you for your interest in contributing to the Personal AI Assistant project!
All contributions, whether big or small, are welcome.

## How to Contribute

1. Fork the repository to your GitHub account.
2. Clone the forked repository to your local machine.
3. Create a new branch for your changes:
4. Make your changes with clean, readable, and well-documented code.
5. Commit your changes with a clear and meaningful commit message.
6. Push your branch to your forked repository.
7. Open a pull request to the main branch of this repository.

## Code Style Guidelines
- Follow PEP8 standards for Python code.
- Use meaningful variable and function names.
- Keep functions small and focused.
- Add comments where the logic is not obvious.

## Reporting Issues
If you find a bug or want to suggest a feature:
- Check existing issues before creating a new one.
- Provide clear steps to reproduce bugs.
- Include relevant logs or screenshots if applicable.

## Pull Request Guidelines
- Reference the related issue in your pull request (if applicable).
- Ensure your code does not break existing functionality.
- Make sure the project runs successfully before submitting.
